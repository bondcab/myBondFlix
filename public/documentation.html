James Bond API

This API will store information on all 27 of the James Bond films released since 1962





Business Logic URL HTTP Method Request body data format Response body data format
Get a list of James Bond films /movies GET None A JSON object holding data about all the movies
Get data about a single film by title /movies/[title] GET None A JSON object holding data about a single James Bond film, title, director, actor, year. Example: { title: 'Casino Royale', director: 'Martin Campbell', actor: 'Daniel Craig' year: '2006' }
Get data about a director by directors name /movies[director] GET None A JSON object holding information about the director: { name: 'Martin Campbell', age: 79, DOB: '24/10/1943' }
Register a user /users POST A JSON object holding data about a user to register: { name: 'Tom Smith', favouriteFilms: [] } A JSON object holding data about the user that was added, including an ID: { id: 3, name: 'Tom Smith', favouriteFilms: [] }
Update user information /users[username] PUT None A text message indicating the student name and username that was updated
Users ability to save favourite Bond films /movies/[favourites]/[title] POST A JSON object holding data on the students favourite Bond films: { title: 'Casino Royale'; } A JSON object holding data on the students favourite Bond films: { title: 'Casino Royale'; }
Users ability to remove title from favourite Bond films /movies/[favourites]/[title] DELETE None A text message indicating whether the film was succesfully removed from favourites list
Users ability to deregister /users DELETE None A text message indicating whether user unregistered succesfully
